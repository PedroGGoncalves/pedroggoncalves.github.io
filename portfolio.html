Novo Scanner

Software do Novo Produto

Desenvolvido com Python, OpenCV, Open3D, CloudCompare, C++ e ImageJ

O objetivo deste projeto foi desenvolver o software para o novo produto da empresa: um Scanner Intraoral próprio. Diversas etapas foram conduzidas para garantir o funcionamento, calibração e processamento adequados do dispositivo. Entre as atividades realizadas destacam-se:

  • Medição da corrente elétrica e calibração do sensor.
  • Estimativa do ponto focal ideal e correção de oclusão das arcadas dentárias.
  • Atualização do firmware e integração de diferentes módulos de software.
  • Validação das imagens utilizando múltiplas ferramentas.

O projeto contou com a participação de uma equipe multidisciplinar e o uso de várias tecnologias para alcançar a qualidade necessária nas imagens e no funcionamento do scanner.

Imagem com Stitching

Melhoria do Algoritmo de Unificação de Imagens

Desenvolvido com Java, C++, ImageJ, Excel e GitHub

O algoritmo anterior era mais lento e apresentava dificuldades de manutenção e geração da imagem unificada. Para otimizar o processo, foi implementado um método baseado em ICP (Iterative Closest Point), tornando-o mais eficiente e preciso. Tanto a parte em C++ quanto a interface em Java foram atualizadas, e diversos testes foram realizados para garantir a superioridade do novo algoritmo.
